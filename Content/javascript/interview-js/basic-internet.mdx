### Что такое cookie & кэш. В чем разнциа?

* `cookie` - это небольшие текстовые файлы, которые хранятся на стороне пользователя на компьютере и создаются, когда пользователь заходит на веб-сайты. Они хранят информацию о работе пользователя на сайте. Статистика, аунтентификация, персональные данные настроек и тд..
По умолчанию, максимальный размер файла cookie равен 4 096 байт.
* `Кеш` - Используется для более быстрой загрузки сайтов. Кеш хранит картинки, видео, текстовую информацию и так далее.  Данные с самого сайта по сути. При повторном посещении сайта браузер проверяет содержимое сайта, а потом выполняет загрузку лишь новой информации.


### Что такое HTTP протокол 
* HTTP - обеспечивает возможность коммуникации между вашим браузером и удалённым сервером. 

### Какие есть методы HTTP-запросов 

* POST - используется для отправки тела. `JSON` или `HTML`.
* GET - Метод для запроса данных. Запрашивает ресурс - то, что идет после доменного имени.
* HEAD - запрашивает ресурс так же, как и метод GET, но без тела ответа.
* DELETE - удаляет указанный ресурс

### Какие есть коды ответов HTTP
* Информационные 100 - 199.
* Успешные 200 - 299.
* Перенаправления 300 - 399.
* Клиентские ошибки 400 - 499.
* Серверные ошибки 500 - 599.
